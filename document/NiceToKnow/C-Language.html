A Brief History of C Language

  • Influenced by BPCL, ALGOL 68, assembly, FORTRAN
  • Primarily used for development of UNIX
  • Reduced keywords from BPCL
  • Become popular after publishing the book : K&R C (1978)
  • Standard : K&R C, ISO C, C99, C11, C17

C Popularity in Last 2 Decades

Why C is so popular?

  • Concise Syntax
  • Power to control
  • Performance (~30 time faster than Python)
  • Portable (Almost every type of CPU and OS)
  • Play well with others

Why not to use C?

  • Power to control : Memory management hell
    • ~30% of bugs from C/C++ are caused by memory management.
    • The most security issues are caused by memory management.
  • Too primitive (~3 time more LoC, ~5-10 time more dev. time than Python)
